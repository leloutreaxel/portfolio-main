Linux

Vous avez découvert Linux et vous souhaitez aller plus loin ? Alors vous êtes au bon endroit ! Et si vous avez déjà utilisé Linux pour votre poste de travail, c’est super, mais maintenant vous allez pouvoir passer à l’étape supérieure… Dans ce cours, vous allez apprendre les bases de l’administration d’un système Linux, c’est-à-dire apprendre à l'utiliser pour un serveur. Aujourd’hui, la majorité des services web est hébergée sur un Linux. Comprendre comment cela fonctionne, concrètement, vous ouvrira des portes !
